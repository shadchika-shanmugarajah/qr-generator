# QR Code Generator

Generate QR codes for any URL—perfect for Google Drive links, websites, and more.

## How to Use

1. **Open** `index.html` in your web browser (double-click the file).
2. **Enter or paste** your URL in the input field (e.g., a Google Drive link).
3. **Click** "Generate QR Code" to create the QR code.
4. **Click** "Download QR Code" to save it as a PNG image.

## Supported Links

- Google Drive links (files, folders)
- Any website URL
- MP3, MP4, PDF, or other file links

## Requirements

- A modern web browser (Chrome, Firefox, Edge, Safari)
- No installation needed—works offline after first load
